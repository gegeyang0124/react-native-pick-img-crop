  ###  安装组件：
  npm i --save react-native-pick-img-crop
  
  ### ios安装
  #### pod 安装
  将 pod 'RNPickImgCrop', :path => '../node_modules/react-native-pick-img-crop' 拷贝进Podfile即可；
  #### 传统安装
  react-native link react-native-pick-img-crop
